Module scheduler

A module allowing for functions to be scheduled based on certain conditions.

Functions

Functions

sleep (seconds)
Sleeps the running coroutine until seconds has passed. Will not work on the main thread.

Parameters:

  • seconds number How many seconds to sleep for.

Returns:

    nil
after (seconds, fct, group)
Run fct after seconds has passed. Returns a function which cancels this function.

Parameters:

  • seconds number How many seconds to wait before running fct.
  • fct function The function to run after the time has passed.
  • group string or nil (Optional) The group the function should be in. Defaults to "default"

Returns:

    function a function which cancels this function.
before (seconds, fct, cancelFct, group)
Run fct until seconds has passed. Returns a function which cancels this function.

Parameters:

  • seconds number How many seconds to wait until fct should stop being run.
  • fct function The function to continuously run before the time has passed.
  • cancelFct function or nil (Optional) A function to run if this one is cancelled.
  • group string or nil (Optional) The group the function should be in. Defaults to "default".

Returns:

    function a function which cancels this function.
_until (conditionFct, fct, cancelFct, group)
Runs fct until conditionFct returns a truthy value or the function returned is called.

Parameters:

  • conditionFct function A function supplying the condition that will stop fct.
  • fct function The function to run continuously while conditionFct returns a truthy value.
  • cancelFct function or nil (Optional) A function to run if this one is cancelled.
  • group string or nil (Optional) The group the function should be in. Defaults to "default".

Returns:

    function a function which cancels this function.
when (conditionFct, fct, cancelFct, group)
Runs fct when conditionFct returns a truthy value or the function returned is called without a parameter.

Parameters:

  • conditionFct function A function supplying the condition that will run fct.
  • fct function The function to run when conditionFct returns a truthy value.
  • cancelFct function or nil (Optional) A function to run if this one is cancelled.
  • group string or nil (Optional) The group the function should be in. Defaults to "default".

Returns:

    function A function which cancels this function. Takes a parameter specifying if fct should not be run.
every (seconds, fct, cancelFct, group)
Runs fct every seconds seconds.

Parameters:

  • seconds number The number of seconds to wait between running fct.
  • fct function The function to run every seconds seconds.
  • cancelFct function or nil (Optional) A function to run if this one is cancelled.
  • group string or nil (Optional) The group the function should be in. Defaults to "default".

Returns:

    function a function which cancels this function.
everyCondition (conditionFct, fct, risingFct, fallingFct, cancelFct, group)
Runs fct every time conditionFct returns a truthy value.

Parameters:

  • conditionFct function The function supplying the condition that will run fct.
  • fct function or nil (Optional) The function to run every time conditionFct returns a truthy value.
  • risingFct function or nil (Optional) The function to run every time conditionFct goes from false to true.
  • fallingFct function or nil (Optional) The function to run every time conditionFct goes from true to false.
  • cancelFct function or nil (Optional) A function to run if this one is cancelled.
  • group string or nil (Optional) The group the function should be in. Defaults to "default".

Returns:

    function a function which cancels this function.
pause (group)
Pause the given group.

Parameters:

  • group any The key of the group to pause. Can technically be any value.

Returns:

    nil
resume (group)
Resumes the given group.

Parameters:

  • group any The key of the group to resume. Can technically be any value.

Returns:

    nil
update (dt)
Updates all of the functions in the scheduler, checking if any need to run and running them if necessary.

Parameters:

  • dt number The time between the last update and this one.

Returns:

    nil
